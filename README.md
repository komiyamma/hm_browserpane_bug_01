# 「空のブラウザ枠」に次のようなマクロを送信してみる

```
setbrowserpaneurl "javascript:window.scrollBy(0,100);";
```

すでにブラウザ枠どころか秀丸エディタが異常状態になっている  
(何かの終了を待っていてマクロが終了しない状態だと思われる)

# 異常状態になっている時は、ブラウザ枠にて右クリックすると次のようなWebView2の元来のコンポーネントが出るのでわかりやすい
